3.1 Diferencia de Promedios

Utilizaremos como ejemplo los datos de sobre medidas del cuerpo para hombres y mujeres de diferentes edades

# Se lee la base de datos desde el enlace y se almacena en datos
url <-
'https://raw.githubusercontent.com/fhernanb/datos/master/medidas_cuerpo'
datos <- read.table(file = url, header = T)
#Se extraen datos de hombres y mujeres
hombres <- datos[datos$sexo == "Hombre", ]
mujeres <- datos[datos$sexo == "Mujer", ]
dim(hombres)
## [1] 18 6
dim(mujeres)
## [1] 18 6
str(hombres)

## $ altura: num 188 174 176 180 168 ...
## $ sexo : chr "Hombre" "Hombre" "Hombre" "Hombre" ...

Note que tanto para hombres como para mujeres, se tienen 18 datos con 6 variables diferentes a escoger: edad, peso, altura, sexo, muñeca y biceps Para calcular un IC de 95% para la diferencia promedio de altura de los hombres y con respecto a las mujeres, dato que el tamaño de la muestra es pequeño, se usará un t.test.

# Media de hombres menos media de mujeres
t.test(hombres$altura, mujeres$altura,conf.level = 0.95)$conf.int
## [1] 10.05574 20.03315
## attr(,"conf.level")
## [1] 0.95
# Media de mujeres menos media de hombres
t.test(mujeres$altura, hombres$altura, conf.level = 0.95)$conf.int
## [1] -20.03315 -10.05574
## attr(,"conf.level")
## [1] 0.95

3.2 Cociente de varianzas

Con estos mismo datos, se pueden realizar otras pruebas, como el cociente de varianzas. Usasemos una función llamada var.test del paquete stests, que seguidamente se puede instalar con el siguiente código (solo debe ejecutarse una vez para instalar):

if (!require('devtools')) install.packages('devtools')
devtools::install_github('fhernanb/stests', force=TRUE)

La función var.test también existe en el paquete stats, por lo que si se desea utilizar dicha función, se va a generar un conflicto. Con las siguientes instrucciones de llamado, se le puede indicar a R cuál paquete se desea usar:

stats::var.test() # Para usar la fución del paquete stats
stests::var.test() # Para usar la fución del paquete stests

var.test del paquete stests es una versión ampliada de var.test del paquete stats.

¿Cuál sería la diferencia entre una y otra versión? Recuerde que: stats::var.test() sólo sirve para 1 población, mientras que stests::var.test() sirve para 1 o 2 poblaciones.

# IC para el cociente de varianzas
stests :: var.test(hombres$altura, mujeres$altura,conf.level =
0.95)$conf.int
## [1] 0.2327398 1.6632830
## attr(,"conf.level")
## [1] 0.95
# IC para el cociente de desviaciones estándar
sqrt(stests :: var.test(hombres$altura,
 mujeres$altura,conf.level = 0.95)$conf.int)
## [1] 0.4824311 1.2896833
## attr(,"conf.level")
## [1] 0.95

3.3 Diferencia de proporciones

El siguiente y último caso se trabajará para diferencia de proporciones, donde se estará comparando la proporción de hombres que tienen menos de 30 años con respecto a la de mujeres que tienen menos de 30 años.

# La función wich permite realizar los conteos
w <- which((hombres$edad <= 30))
w
## [1] 7 9 10 12 13 15 16
exitos.hombres <- length(which(hombres$edad <= 30))
exitos.mujeres <- length(which(mujeres$edad <= 30))
exitos.hombres
## [1] 7
exitos.mujeres
## [1] 15
length(hombres$edad)
## [1] 18

Para construir intervalos de confianza bilaterales para la proporción a partir de la función prop.test es necesario definir 3 argumentos:

  • x: Vector con el conteo de éxitos de las dos muestras.
  • n: Vector con el número de ensayos de ambas muestras.
  • conf.level: Nivel de confianza.

Seguidamente se construye un IC para la diferencia de proporciones entre hombres y mujeres menores de 30 años, con un nivel de confianza de 90%

# Proporción de hombres menos proporción de mujeres
prop.test(x = c(exitos.hombres, exitos.mujeres), n =
c(length(hombres$edad), length(mujeres$edad)), conf.level =
0.90)$conf.int
## [1] -0.7379022 -0.1509867
## attr(,"conf.level")
## [1] 0.9
# Proporción de mujeres menos proporción de hombres
prop.test(x = c(exitos.mujeres, exitos.hombres), n =
c(length(mujeres$edad), length(hombres$edad)), conf.level =
0.90)$conf.int
## [1] 0.1509867 0.7379022
## attr(,"conf.level")
## [1] 0.9
