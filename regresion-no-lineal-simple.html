Ejemplo 1:

Considere los datos de la siguiente tabla:

X Y
18 19

  • Escoja y justifique un modelo de regresión para 𝑌 como función de 𝑋, sabiendo que la variable 𝑋 por su naturaleza, toma valores menores que 20.

  • Determine el valor esperado de \(Y\) cuando \(x = 17\)

  • Determine un Intervalo de Confianza para los parámetros \(\alpha\) y \(\beta\), según el modelo escogido previamente.

Para escoger el modelo, se recomienda primeramente construir un gráfico de dispersión para ver la tendencia de los datos:

# Creamos la tabla de valores
X <- c(18, 19, 19.5, 19.7, 19.9) 
Y <- c(19, 40, 79, 130, 397)

datos <- cbind(X, Y) # se juntan las columnas
datos.df <- as.data.frame(datos) # se crea un objeto dataframe
names(datos.df) # verificamos los nombres de las columnas
## [1] "X" "Y"
# Se crea un gráfico de dipsersión para ver de manera exporatoria los datos
g1 <- ggplot(datos.df, aes(X, Y))
g1 + geom_point(shape = 21, width = 0.3, alpha = 0.4, fill = "blue", size = 1, stroke = 1)
## Warning in geom_point(shape = 21, width = 0.3, alpha = 0.4, fill = "blue", :
## Ignoring unknown parameters: `width`

De los datos anteriores, se considera que un buen modelo de RNLS para los datos, puede ser el Modelo Hiperbólico, es decir:

Modelo Transformación
\(y = \frac{x}{\alpha x + \beta}\) Aplicando logaritmo natural:
\(\frac{1}{y} = a + \beta \cdot \frac{1}{x}\)
Condiciones Cambios de variable:
\(y_1 = \frac{1}{y}; \; x_1 = \frac{1}{x}; \; \beta_0 = \alpha; \; \beta_1 = \beta\)
Posee asíntotas verticales y horizontales que no necesariamente son los ejes Modelo lineal obtenido:
\(y_1 = \beta_0 + \beta_1 \cdot x_1\)

Foto

Lo anterior debido a que se percibe del gráfico que conforme \(x \rightarrow 20^-\), los valores de \(y\) crecen rápidamente, semejando en \(x = 20\) una asíntota vertical. Otros modelos como el exponencial se descarta por no tener asíntotas verticales y el potencial o recíproco solo tienen asíntota vertical en \(x = 0\)

Procedemos con las modificaciones en la tabla de datos, para transformar el modelo en un caso lineal, es decir, se hace una linealización de los datos:

Según la transformación recomendada, se debe hacer dos cambios de variable: \(X_2 = \frac{1}{X}\) y \(Y_2 = \frac{1}{Y}\), para luego suministrarle estos valores a la función lm de R para crear una MRL:

X2 <- 1/X
Y2 <- 1/Y

datos2 <- cbind(X2, Y2)
datos2.df <- as.data.frame(datos2)
modelolm <- lm(Y2 ~ X2, data = datos2.df)
modelolm
## 
## Call:
## lm(formula = Y2 ~ X2, data = datos2.df)
## 
## Coefficients:
## (Intercept)           X2  
##     -0.4698       9.4032

Del modelo anterior se tiene que:

\(a = -0.4698\) y \(\beta = 9.4032\)

Por lo que haciendo nuevamente la transformación, el modelo buscado sería:

\[ y = \frac{x}{-0.4698x + 9.4032} \] Los siguiente códigos ayudarán a mostrar gráficamente el modelo obtenido con respecto a los datos del gráfico de dispersión:

# Se genera una secuencia de puntos, para simular continuidad en la gráfica

yo <- xo/(-0.4698*xo + 9.4032)
# Graficamos dispersión y el modelo de ajuste en un mismo plot
plot(X,Y, main = "Modelo de RNLS f(x)=-x/(0.4698*x + 9.4032)", col ="darkblue", lwd = 2)
lines(xo, yo, type = "l")

Ejemplo 2:

Considere los datos de la siguiente tabla:

x Y
1 4
2 7
4 20
6 40

A partir de los datos, estime el coeficiente \(\beta\) de la ecuación de regresión \(y = 3 + x^\beta\), transformando el modelo a un modelo a un modelo de Regresión Lineal Simple mediante el método de mínimos cuadrados.

Para escoger el modelo, se recomienda primeramente construir un gráfico de dispersión para ver la tendencia de los datos:

# Creamos la tabla de valores
X <- c(1, 2, 4, 6) 
Y <- c(4, 7, 20, 40) 
datos3 <- cbind(X, Y) # se juntan las columnas
datos3.df <- as.data.frame(datos3) # se crea un objeto dataframe
names(datos3.df) # verificamos los nombres de las columnas
## [1] "X" "Y"
# Se crea un gráfico de dipsersión para ver de manera exporatoria los datos
g1 <- ggplot(datos3.df, aes(X, Y))
g1 + geom_point(shape = 21, alpha = 0.4, fill = "blue", size = 1, stroke = 1)

De los datos anteriores, se considera que un buen modelo de regresión No lineal para los datos, puede ser el Modelo Potencial, es decir: | Modelo | Transformación | | \(y = a \cdot x^\beta\) | Aplicando logaritmo natural:
\(ln(y) = ln(\alpha) + \beta \cdot ln(x)\) | | Condiciones: | Cambios de variable:
$ y_1 = ln(y); ; x_1 = ln(x); ; _0 = ln(); ; _1 = $ | | Si \(\beta > 0\), el modelo pasa por \((0,0)\) | Modelo lineal obtenido:
\(y_1 = \beta_0 + \beta_1 \cdot x_1\) |

Foto

Note que el modelo propuesto, discrepa del modelo base en una constante, pero esto no limita a que se pueda linealizar, sin embargo, no de puede adoptar las recomendaciones de la imagen tal cuál se detallan, sino que podemos hacer la transformación como sigue.

\[ y = 3 + x^\beta \\ y-3 = x^\beta \\ ln(y-3) = ln(x^\beta) \\ ln(y-3) = \beta \; ln(x) \\ \] Procedemos con las modificaciones en la tabla de datos, para transformar el modelo en un caso lineal:

Según la transformación recomendada y adaptada a esta variante del modelo, se debe hacer los siguientes cambios de variable: \(X_2 = \ln(x), Y_2 = \ln(y-3), b_0 = \ln(1) = 0, b_1 = \beta\) para luego suministrarle estos valores a la función lm de R para crear una MRLS. Note que en este caso como \(b_0 = 0\) se concluye que el modelo pasa por el origen, es decir, por el punto \((0,0)\) y eso se verá reflejado en la nueva tabla como se muestra a continuación:

# Se modifican los datos de la tabla original

modelolm

## 
## Coefficients:
##    X2  
## 2.024
