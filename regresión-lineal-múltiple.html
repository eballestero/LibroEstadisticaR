13.1 Regresión Lineal Múltiple (RLM)

Ejemplo 1

En la siguiente tabla, se registran las siguiente observaciones:

\(X_1\) \(X_2\) \(y\)
0 1 1
1 0 4
1 -1 6
0 0 3
-1 1 0

  • Estime la ecuación de regresión de \(Y\) en función de \(X_1\), \(X_2\).

  • Calcule el coeficiente de determinación y correlación del modelo.

  • Determine el valor esperado aproximado de \(Y\) cuando $X_1 = 2 $ y \(X_2 = 3\)

Solución

Primeramente se genera la matriz asociada al sistema:

X <- matrix(c(1,0,1,1,1,0,1,1,-1,1,0,0,1,-1,1), nrow = 5, byrow = TRUE)
X
##      [,1] [,2] [,3]
## [1,]    1    0    1
## [2,]    1    1    0
## [3,]    1    1   -1
## [4,]    1    0    0
## [5,]    1   -1    1
# La instrucción genera un vector columna
Y <- cbind(c(1,4,6,3,0))
Y
##      [,1]
## [1,]    1
## [2,]    4
## [3,]    6
## [4,]    3
## [5,]    0

Seguidamente, usando el teorema siguiente:

\[ b = (X^t \; X)^{-1} (X^t \;Y) \] Para el trabajo en R tome en cuenta las siguientes instrucciones: Sea M una matriz de orden n, así

  • solve(M): Calcila la matriz inversa de M

  • t(M): Calcula la matriz transpuesta de M

  • %*%: Corresponde a la multiplicación matricial

Para calcular las estimaciones buscadas, hacemos el siguiente cálculo:

estimaciones.b1 <- solve(t(X) %*% X) %*% (t(X) %*% Y)
estimaciones.b1
##      [,1]
## [1,]    3
## [2,]    1
## [3,]   -2

Así, el modelo de regresión buscado sería: \[ y = 3 + X_1 -2X_2 \] Otra forma de estimar el modelo, es haciendo uso de la función lm que se usó para el modelo de Regresión Lineal Simple, como se muestra a continuación:

modelo <- lm(Y ~ X[ ,2]+X[ ,3])
modelo
## 
## Call:
## lm(formula = Y ~ X[, 2] + X[, 3])
## 
## Coefficients:
## (Intercept)       X[, 2]       X[, 3]  
##           3            1           -2

Veamos la calidad del modelo, mediante su coeficiente de determinación y de correlación:

# Coefieciente de determinación
R2 <- summary(modelo)$r.squared
## Warning in summary.lm(modelo): essentially perfect fit: summary may be
## unreliable
R2
## [1] 1
# Coeficiente de correlación
R <- sqrt(R2)
R
## [1] 1

Seguidamente, se crean una función para realizar las estimaciones

modeloRLM1 <- function(x1,x2){
 y <- 3 + x1 - 2*x2
 return(y)
}
modeloRLM1(2,3)
## [1] -1

Ejemplo 2 Se pustula que la altura de un bebéé \((y)\) tiene una relación lineal con su edad en días \((x_1)\), su altura al nacer en cm \((x_2)\), su peso en kg al nacer \((x_3)\) y el aumento en tanto por ciento de su peso actual respecot de su peso al nacer \((x_4)\).

13.2 Regresión No Lineal Múltiple (RNLM)

Ejemplo 1

La tabla a trabjar es la siguiente:

\(x\) \(y\) \(z\)
3 5 -46
8 9 -110
13 4 10
16 6 -12
20 2 64

Modelo a trabajar: $ z = _0 + _1 x + _2 y + _3 y^2$

Se crea la tabla en R de la siguiente manera:

M <- matrix(c(3,5,8,9,13,4,16,6,20,2), nrow = 5, byrow = TRUE)
M
##      [,1] [,2]
## [1,]    3    5
## [2,]    8    9
## [3,]   13    4
## [4,]   16    6
## [5,]   20    2
# La instrucción genera un vector columna, que representa el vector solución
z <- cbind(c(-46,-110,10,-12,64))
z
##      [,1]
## [1,]  -46
## [2,] -110
## [3,]   10
## [4,]  -12
## [5,]   64

Para linealizar el modelo, se debe hacer el siguiente cambio de variable:

  • \(x_3\) = \(y^2\)

  • \(x_1\) = \(x\)

  • \(x_2\) = \(y\)

  • \(z_2\) = \(z\)

  • \(b_0\) = \(\beta_0\)

  • \(b_1\) = \(\beta_1\)

  • \(b_2\) = \(\beta_2\)

  • \(b_3\) = \(\beta_3\)

Transformamos la tabla original, para que se ajuste al modelo de Regresión Lineal Múltiple:

x <- cbind(c(1,1,1,1,1), M[ ,c(1, 2)], M[ ,2]^2 )
x
##      [,1] [,2] [,3] [,4]
## [1,]    1    3    5   25
## [2,]    1    8    9   81
## [3,]    1   13    4   16
## [4,]    1   16    6   36
## [5,]    1   20    2    4

Con esta nueva matriz, resolvemos el siguiente modelo: \[ z_2 = b_0 + b_1 x_1 + b_2 x_2 + b_3 x_3 \] como si fuese un caso de (Regresión Lineal Múltiple) RLM, de la siguiente manera:

estimaciones.b <- solve(t(x) %*% x) %*% (t(x) %*% z)
estimaciones.b
##      [,1]
## [1,]    2
## [2,]    4
## [3,]   -7
## [4,]   -1

Recordenmos también que el modelo se puede obtener usando la funión lm:

modelo1 <- lm(z ~ x[,2] + x[,3] + x[,4])
modelo1
## 
## Call:
## lm(formula = z ~ x[, 2] + x[, 3] + x[, 4])
## 
## Coefficients:
## (Intercept)       x[, 2]       x[, 3]       x[, 4]  
##           2            4           -7           -1

Por tanto, el modelo de Regresión Lineal Múltiple buscado, sería: \[ \hat{z} = 2 + 4x - 7y - y^2 \] Evaluamos el modelos para los valores de \(x = 2\) y \(y = 7\)

b <- estimaciones.b
modeloRNLM1 <- function(x,y){
 z <- b[1,] + b[2,]*x + b[3,]*y + b[4, ]*y^2
 return(z)
}

modeloRNLM1(2,7)
## [1] -88

Ejemplo 2: La tabla a trabjar es la siguiente:

\(x\) \(y\) \(z\)
1 0.5 0.06
1.5 1 6.7
2 2 512
1 2 64

Modelo a trabajar: \(z = \beta_0 x^{\beta_1} y^{\beta_2}\)

Se crea la tabla en R de la siguiente manera:

M2 <- matrix(c(1,0.5,1.5,1,2,2,1,2), nrow = 4, byrow = TRUE)
M2
##      [,1] [,2]
## [1,]  1.0  0.5
## [2,]  1.5  1.0
## [3,]  2.0  2.0
## [4,]  1.0  2.0
# La instrucción genera un vector columna, que representa el vector solución
z <- cbind(c(0.06,6.7,512,64))
z
##        [,1]
## [1,]   0.06
## [2,]   6.70
## [3,] 512.00
## [4,]  64.00

Se procede a la siguiente transformación del modelo original, con el fin de buscar una linealización del mismo:

\[ \ln(z) = \ln(\beta_0) + \beta_1 \ln(x) + \beta_2 \ln(y) \] Para linealizar el modelo, se debe hacer el siguiente cambio de variable:

  • \(b_0\) = \(\ln(\beta_0)\)

  • \(b_1\) = \(\beta_1\)

  • \(b_2\) = \(\beta_2\)

  • \(x_1\) = \(\ln(x)\)

  • \(x_2\) = \(\ln(y)\)

  • \(z_2\) = \(\ln(z)\)

Tranformamos la tabla original, para que se ajuste al modelo de RLM:

x2 <- cbind(c(1,1,1,1), log(M2[ ,c(1, 2)]))
x2
##      [,1]      [,2]       [,3]
## [1,]    1 0.0000000 -0.6931472
## [2,]    1 0.4054651  0.0000000
## [3,]    1 0.6931472  0.6931472
## [4,]    1 0.0000000  0.6931472
z2 <- log(z)
z2
##           [,1]
## [1,] -2.813411
## [2,]  1.902108
## [3,]  6.238325
## [4,]  4.158883

Con esta nueva matriz, resolvemos el siguiente modelo:

\[ z_2 = b_0 + b_1 x_2 +b_2 x_2 \] como si fuese una caso de RLM, de la siguiente manera:

estimaciones.b <- solve(t(x2) %*% x2) %*% (t(x2) %*% z2)
estimaciones.b
##           [,1]
## [1,] 0.6744198
## [2,] 3.0078412
## [3,] 5.0260234

Recordemos que estos coeficientes se pueden calcular con la función lm de R:

modelo1 <- lm(z2 ~ x2[,2] + x2[,3])
modelo1
## 
## Call:
## lm(formula = z2 ~ x2[, 2] + x2[, 3])
## 
## Coefficients:
## (Intercept)      x2[, 2]      x2[, 3]  
##      0.6744       3.0078       5.0260

Recordemos los cambios de variables que se utilizaron para las estimaciones de los parámetros:

  • \(b_0\) = \(\ln(\beta_0)\)

  • \(b_1\) = \(\beta_1\)

  • \(b_2\) = \(\beta_2\)

De esta manera, para obterner el valor bucado, se precede con el siguiente despeje:

  • \(\beta_0\) = \(e^{b_0}\)

Hacemos este cálculo en R para obtener nuevamente el vector con los valores de las estimaciones para los betas:

b <- estimaciones.b
beta <- c(exp(b[1, ]) , b[2, ], b[3, ]) 
beta
## [1] 1.962894 3.007841 5.026023

Por tanto, el modelo de RNLM buscado, sería:

\[ z = (1.962894)x^{3.007841} y^{5.026023} \] Evaluamos el modelos para los valores de \(x = 2\) y \(y = 7\)

b <- estimaciones.b
modeloRNLM2 <- function(x1,x2){
 z <- b[1,]* x1^b[2,]*x2^b[3,]
 return(z)
}
modeloRNLM2(2,7)
## [1] 95909.87
